# Project Atlas: The Macro Topology

이 문서는 프로젝트의 **거시적 지형도(World Map)**입니다.
세부적인 지식의 연결은 각 **책(Book)** 내부의 **시냅스(Synapses)**를 따라 탐색하십시오.

## 🌏 Domain Clusters (지식의 대륙)

### 1. 🎵 Player Cluster
> **Context**: 영상 재생, 가사 렌더링, 싱크 조절 등 사용자 경험의 핵심 영역.
- **Entry Book**: `domains/player_ui.md` (플레이어의 구조)

### 2. 🛠️ Editor Cluster
> **Context**: 자막 제작, 데이터 수정, 어드민 기능 영역.
- **Entry Book**: `domains/editor_ui.md` (편집기 워크스페이스)

### 3. 💾 Tech & Data Cluster
> **Context**: 데이터베이스 스키마, 배포 인프라, 공통 유틸리티.
- **Entry Book**: `tech/db_schema.md` (데이터 구조)

---

## 🚀 How to Traverse (탐색 가이드)

1. **Select Cluster**: 작업하려는 영역(Domain)을 위에서 고르십시오.
2. **Enter Book**: 해당 클러스터의 **Entry Book**을 읽으십시오.
3. **Follow Synapses**: 책 내부에 적힌 `🔗 Synapses` 섹션을 보고, **연관된 지식(0.1 ~ 0.3)**으로 이동하여 맥락을 파악하십시오.
