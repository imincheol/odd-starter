# 📚 ODD Library (지식의 보고)

> **"지식은 쌓이는 것이 아니라, 연결되는 것입니다."**

이곳은 **ODD Starter Kit** 프로젝트의 **살아있는 지식(Living Knowledge)**을 보관하는 도서관입니다.
이곳의 모든 문서는 **현재 유효한 최신 상태(Current State)**를 유지해야 합니다. (v.s. History)

## 📖 Bookshelf (서가)

### 1. 🏛️ General (개요 및 정체성)
- **Overview**: 이 프로젝트는 무엇이며 왜 존재하는가?
- **Rules**: 팀 그라운드 룰, 컨벤션.

### 2. 💼 Domain (비즈니스 로직)
- **Order System**: 오더 기반 개발 프로세스 심층 분석.
- **Memory Model**: Library와 History의 이원화 구조에 대한 정의.

### 3. 🛠️ Tech (기술 스택)
- **Tech Stack**: 사용 중인 프레임워크와 라이브러리.
